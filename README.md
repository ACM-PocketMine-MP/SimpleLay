### Description
With this plugin, you can lay down wherever you want!

### Commands
| Command | Description | Permission |
| --- | --- | --- |
| ```/lay``` | Lay on a Block | - |

### Issues
Known Issues: 
- The player does not fall if there are no blocks under the player (No Gravity) WELP!
